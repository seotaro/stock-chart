## Install

```bash
yarn
```

## Run

create .env file

```Dotenv
REACT_APP_TWELVEDATA_API_KEY=xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
REACT_APP_TWELVEDATA_API_INTERVAL=30000
```

```bash
yarn start
```

## Notes

- Financial Data API: [twelvedata](https://twelvedata.com/)
- Chart: [Highcharts](https://www.highcharts.com/)
